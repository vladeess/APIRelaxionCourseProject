# APIRelaxionCourseProject
Курсовой проект по дисциплине "Распределённые информационные системы" на тему:"Распределённая система решения СЛАУ методом релаксации.
Репозиторий содержит 3 проекта:
-1 проект - нераспределённое решение СЛАУ методом релаксации.
-2 проект - распределённое решение СЛАУ методом релаксации.
-3 проект - проект, предназначенный для создания порта и подключения по нему распределенного решения и передача матрицы.
